# Profile Image

Drop your profile photo here, named exactly:

    profile.jpg

Hugo serves everything in `/static/` at the site root, so
`static/images/profile.jpg` becomes `https://yourdomain/images/profile.jpg`,
which matches the `imageUrl` in hugo.yaml.

Recommended specs:
- Square crop (1:1 ratio)
- Minimum 400×400 px (displays at 148×148 px, but 2× for retina)
- JPEG or WebP, keep under 150 KB
